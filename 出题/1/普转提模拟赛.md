# 普转提模拟赛

 __dragon_

## 七七(qiqi.cpp/.in/.out)

#### 时间限制：1s 空间限制：64MB

今天，白先生又让七七出去采药了。七七到了野外，看到了 $n$ 种药草。每种草药有 $a_i$ 颗，每株价值为 $v_i$ 。白先生让七七采集药草的总价值尽量大，但是七七的小包只够装 $m$ 株药草。七七不太聪明，她想请你帮忙。请问，七七采集的药草的总价值最大是多少？

### 输入格式
第一行两个整数：$n$ 和 $V$ 。

接下来 $n$ 行，每行两个正整数， $a_i$ 和 $v_i$ 。

### 输出格式
一行一个正整数，表示七七采集药草的最大总价值。

### 样例

#### 输入
```
3 5
1 1
4 5
1 4
```

#### 输出
```
24
```

### 提示
对于 $20\%$ 的数据，满足 $n \leq 100$

对于 $50\%$ 的数据，满足 $n \leq 1000,m \leq 2000$

对于所有测试数据，满足 $1 \leq n \leq 5 \times 10^5;1 \leq m \leq 1 \times 10^8;1 \leq a_i,v_i \leq 1 \times 10^8$
